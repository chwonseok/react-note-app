# What's this?

This is Note App built with React.js, which you can

add a note,

search a note,

delete a note,

reset the note app.

Also, this is responsive web app 😉

# Live

https://chwonseok.github.io/react-note-app/

# Dependencies

1. react-icons

   - `npm install react-icons`

2. uniqid

   - `npm install uniqid`

# Hooks used

- `useState()`
- `useEffect()`
